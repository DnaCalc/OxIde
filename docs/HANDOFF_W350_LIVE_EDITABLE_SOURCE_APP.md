## What W350 Delivers

W350 turns DnaOxIde from a static host proof into a basic reviewable browser-driven source app:

```text
render app → type source text → observe dirty state → save temp project copy → edit divergently → reload saved text → inspect visual/DOM/log artifacts
```

## Accepted Scope

W350 may claim:

- browser DOM live source editing is driven by Playwright/Edge;
- the source pane is a real editable `<textarea>` mounted in the app;
- dirty state updates after browser input;
- `window.__DNA_OXIDE_TEST_DRIVER__` exposes snapshots, visual snapshots, event logs, command logs, and interaction injection;
- save writes edited source to a temp project copy under `target/`;
- reload reads the saved temp file and discards divergent unsaved text;
- visual HTML/PNG and JSON artifacts are generated by the automatic loop;
- `npm --prefix apps/dna-oxide run live-edit:check` is the repeatable automatic feedback loop.

## Explicit Non-Claims

W350 does not claim:

- live Tauri/WebView IPC;
- native desktop packaging readiness;
- real OxVba compile/build execution;
- real/native OxVba runtime execution;
- real Immediate evaluation;
- real debug/watch/breakpoint behavior;
- COM runtime invocation;
- real DnaOneCalc product mount;
- full DOM accessibility audit.
